# 🎓 Student Data Analysis Dashboard | Power BI

## 📌 Project Overview
The **Student Data Analysis Dashboard** is an interactive Power BI solution that provides a comprehensive view of school operations, student performance, admissions, fee collection, and management insights. It transforms raw educational data into meaningful visualizations, enabling stakeholders to monitor key metrics and make informed decisions.

## 🎯 Project Objective
The objective of this project is to:
- Monitor student enrollment and admissions.
- Track fee collection and pending payments.
- Analyze school-wise and class-wise performance.
- Visualize student demographics and academic trends.
- Support data-driven decision-making through interactive dashboards.

## 📊 Business Questions (KPIs Answered)

- 🏫 How many schools and students are in the system?
- 👨‍🎓 How many active students are currently enrolled?
- 🆕 How many new admissions were made?
- 💰 What is the total expected fee, collected fee, and pending fee?
- 📈 What is the overall fee collection percentage?
- 🏆 Which schools have the highest student count and fee collection?
- 👦 What is the gender distribution of students?
- 🎂 Which students have birthdays today or this month?
- 📚 How are admissions trending over time?
- 👤 What is the complete profile and fee status of an individual student?
